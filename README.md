# Conway's Game of Life

Play the game of life with given initial state or define your own.

[Live Application](game-of-life-dlzl.surge.sh)

### Project Goals

Application was completed as a freeCodeCamp [challenge](https://www.freecodecamp.org/challenges/build-the-game-of-life).

1. User Story: When I first arrive at the game, it will randomly generate a board and start playing.

2. User Story: I can start and stop the board.

3. User Story: I can set up the board.

4. User Story: I can clear the board.

5. User Story: When I press start, the game will play out.

6. User Story: Each time the board changes, I can see how many generations have gone by.

7. Hint: Here's an explanation of Conway's Game of Life from John Conway himself: https://www.youtube.com/watch?v=E8kUJL04ELA

8. Hint: Here's an overview of Conway's Game of Life with rules for your reference: https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life
